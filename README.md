# 🐱 猫猫给你唱歌~ 🎵

一个超级可爱的猫咪主题本地音乐播放器！让萌萌的猫咪陪伴你的音乐时光~喵喵~

## ✨ 功能特性

🐱 **可爱猫咪主题**
- 粉色系配色，少女心爆棚！
- 动态猫咪ASCII动画，根据播放状态变化表情
- 圆角设计，温馨可爱

📁 **多歌单管理**
- 支持创建多个歌单，每个歌单对应一个音乐文件夹
- 自动记忆上次播放的歌单
- 双击或单击即可切换歌单
- 右键歌单可打开所在目录

🎵 **多种音频格式支持**
- MP3、FLAC、WAV、OGG、M4A、AAC 等主流格式
- 自动读取音频元数据（歌手、标题）
- 智能提示不支持的格式（如APE）

▶️ **播放控制**
- 播放/暂停、上一曲、下一曲
- 带有猫咪爪印风格的大按钮
- 三种播放模式：列表循环 / 单曲循环 / 随机播放
- 播放时自动记录当前歌曲，下次打开继续播放

🔍 **歌曲搜索**
- 实时搜索过滤歌曲
- 支持模糊匹配
- 搜索结果中高亮显示当前播放歌曲

📋 **歌曲排序**
- **拖动排序**：按住歌曲上下拖动调整顺序
- **右键菜单排序**：上移、下移、置顶、置底
- **编辑位置**：输入目标位置数字，精确移动
- 排序结果自动保存

⏱️ **进度控制**
- 实时显示播放进度
- 拖动进度条可跳转到指定位置
- 显示当前播放时间和总时长

🎚️ **音量控制**
- 滑动条快速调节音量
- 数字输入框精确设置（0-100）
- +/- 按钮微调音量（每次±5）
- 自动记忆音量设置

🖥️ **系统托盘**
- 最小化到系统托盘，不占用任务栏
- 双击托盘图标快速恢复窗口
- 右键菜单可控制播放/暂停、下一曲、显示界面、退出

🖱️ **右键菜单**

**歌单右键**：
- 📂 打开歌单所在目录

**歌曲右键**：
- 📂 打开歌曲所在目录
- ⬆️ 上移 / ⬇️ 下移
- ⏫ 移动到最顶 / ⏬ 移动到最低
- ✏️ 编辑位置...（输入目标位置）
- 🗑 删除歌曲（仅从歌单移除，不删除文件）

💾 **智能记忆**
- 自动保存歌单配置和歌曲排序
- 记住上次播放的歌单和歌曲（按文件名）
- 记住音量设置
- 重启后自动恢复到上次状态

## 📖 使用说明

### 基本操作

1. **添加歌单**：点击"➕ 添加歌单"按钮，选择包含音乐文件的文件夹
2. **切换歌单**：在歌单列表中单击或双击即可切换
3. **删除歌单**：选中歌单后点击"🗑 删除"按钮（不会删除实际文件）
4. **播放歌曲**：在歌曲列表中双击歌曲即可播放
5. **搜索歌曲**：在搜索框中输入关键词，实时过滤歌曲列表

### 歌单右键菜单

在歌单列表中**右键点击**歌单：
- **📂 打开歌单所在目录** - 在资源管理器中打开该歌单的文件夹

### 歌曲右键菜单

在歌曲列表中**右键点击**歌曲：
- **📂 打开歌曲所在目录** - 打开文件夹并选中该歌曲文件
- **⬆️ 上移** - 将歌曲向上移动一位
- **⬇️ 下移** - 将歌曲向下移动一位
- **⏫ 移动到最顶** - 将歌曲移到列表第一位
- **⏬ 移动到最低** - 将歌曲移到列表最后一位
- **✏️ 编辑位置...** - 弹出对话框输入目标位置（1-N），歌曲将被移动到该位置
- **🗑 删除歌曲** - 从歌单中移除该歌曲（不会删除实际文件）

> 💡 **提示**：搜索模式下，置顶、置底和编辑位置功能仍然可用，但上移/下移会被禁用。

### 播放控制

- **播放/暂停**：点击中间的播放按钮或双击歌曲
- **上一曲/下一曲**：点击左右两侧的猫咪爪印按钮
- **切换模式**：选择播放模式：🔁列表循环 / 🔂单曲循环 / 🔀随机播放
- **调节进度**：拖动进度条可跳转到指定位置

### 音量调节

- **滑块拖动**：快速调节音量
- **数字输入**：在输入框中输入 0-100 的数值，按回车确认
- **+/- 按钮**：每次点击增加或减少 5 音量
- **自动记忆**：音量设置会自动保存，下次打开时恢复

### 系统托盘功能

- **最小化到托盘**：点击窗口最小化按钮，程序会隐藏到系统托盘
- **恢复窗口**：双击系统托盘图标
- **右键菜单**：右键点击托盘图标可看到菜单：
  - 🐱 显示界面 - 恢复主窗口
  - ⏸ 暂停/播放 - 快速控制播放
  - ⏭ 下一曲 - 切换到下一首歌
  - ❌ 退出 - 完全退出程序

### 播放记忆功能

播放器会自动记录：
- 当前播放的歌单
- 当前播放的歌曲（按文件名记录，即使歌曲顺序改变也能正确找到）
- 当前音量大小

下次打开播放器时，会自动恢复到上次的状态，点击播放按钮即可继续播放上次那首歌。

### 猫咪动画

界面上方有一只可爱的猫咪ASCII动画，它会根据播放状态变化：
- 🐱 **正常状态** - 猫咪在休息 (`/\_/\`)
- 🎵 **播放状态** - 猫咪在唱歌 (`> <`)
- 👂 **暂停状态** - 猫咪在认真聆听 (`o_o`)
- 💤 **空闲状态** - 猫咪会随机打瞌睡 (`-.-`)

## 🎨 主题配色

| 元素 | 颜色 | 用途 |
|------|------|------|
| 主背景 | `#FFF0F5` | 淡粉色，主窗口背景 |
| 次背景 | `#FFE4E1` | 玫瑰色，卡片背景 |
| 强调色 | `#FF69B4` | 热粉色，按钮和选中状态 |
| 深强调色 | `#FF1493` | 深粉色，悬停和激活状态 |
| 文字主色 | `#8B4513` | saddle棕色，主要文字 |
| 文字次色 | `#CD853F` | peru色，次要文字 |

## 📁 配置文件

程序会自动生成 `player_config.json` 文件保存配置：

```json
{
  "playlists": {
    "歌单名称": {
      "folder": "/path/to/music/folder",
      "songs": [
        ["song.mp3", "/full/path/to/song.mp3"]
      ]
    }
  },
  "last_playlist": "上次播放的歌单名称",
  "current_song": {
    "歌单名称": "上次播放的歌曲文件名.mp3"
  },
  "volume": 75
}
```

## 📝 注意事项

- 支持的音频格式：MP3、FLAC、WAV、OGG、M4A、AAC
- APE 格式暂不支持，建议转换为 FLAC 或 MP3
- 歌单只保存文件夹路径和排序信息，不会复制音乐文件
- 如果文件夹被移动或删除，歌单会提示路径不存在
- 系统托盘功能需要 Windows 系统支持
- 歌曲按文件名记录，如果歌曲文件被重命名，播放记忆会失效
