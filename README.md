# 🐱 猫猫给你唱歌~ 🎵

一个超级可爱的猫咪主题本地音乐播放器！让萌萌的猫咪陪伴你的音乐时光~喵喵~

![License](https://img.shields.io/badge/许可-MIT-green?style=for-the-badge)

## ✨ 功能特性

🐱 **可爱猫咪主题**
- 粉色系配色，少女心爆棚！
- 动态猫咪ASCII动画，根据播放状态变化表情
- 圆角设计，温馨可爱

📁 **多歌单管理**
- 支持创建多个歌单，每个歌单对应一个音乐文件夹
- 自动记忆上次播放的歌单
- 双击或单击即可切换歌单

🎵 **多种音频格式支持**
- MP3、FLAC、WAV、OGG、M4A、AAC 等主流格式
- 自动读取音频元数据（歌手、标题）
- 智能提示不支持的格式（如APE）

▶️ **播放控制**
- 播放/暂停、上一曲、下一曲
- 带有猫咪爪印风格的大按钮
- 三种播放模式：列表循环 / 单曲循环 / 随机播放

🔍 **歌曲搜索**
- 实时搜索过滤歌曲
- 支持模糊匹配
- 搜索结果高亮显示

📋 **拖动排序**
- 支持手动拖动歌曲调整顺序
- 播放顺序自动跟随排序变化
- 排序结果自动保存

⏱️ **进度控制**
- 实时显示播放进度
- 拖动进度条可跳转到指定位置
- 显示当前播放时间和总时长

🎚️ **音量控制**
- 滑动条调节音量
- 带猫咪图标的可爱设计

🖥️ **系统托盘**
- 最小化到系统托盘，不占用任务栏
- 双击托盘图标快速恢复窗口
- 右键菜单可控制播放/暂停、下一曲、显示界面、退出

💾 **智能记忆**
- 自动保存歌单配置
- 记住上次播放的歌单
- 重启后自动恢复

## 📦 安装依赖

```bash
pip install -r requirements.txt
```

依赖项：
- `customtkinter` - 现代化的Tkinter主题
- `pygame` - 音频播放引擎
- `mutagen` - 音频元数据读取
- `Pillow` - 图像处理
- `pystray` - 系统托盘图标

## 🚀 运行程序

```bash
python music_player.py
```

## 📖 使用说明

### 基本操作

1. **添加歌单**：点击"➕ 添加歌单"按钮，选择包含音乐文件的文件夹
2. **切换歌单**：在歌单列表中单击或双击即可切换
3. **删除歌单**：选中歌单后点击"🗑 删除"按钮（不会删除实际文件）
4. **播放歌曲**：在歌曲列表中双击歌曲即可播放
5. **搜索歌曲**：在搜索框中输入关键词，实时过滤歌曲列表
6. **调整顺序**：在歌曲列表中按住歌曲拖动即可调整播放顺序

### 播放控制

- **播放/暂停**：点击中间的播放按钮或双击歌曲
- **上一曲/下一曲**：点击左右两侧的猫咪爪印按钮
- **切换模式**：选择播放模式：🔁列表循环 / 🔂单曲循环 / 🔀随机播放
- **调节进度**：拖动进度条可跳转到指定位置
- **调节音量**：使用底部音量滑块调整音量大小

### 系统托盘功能

- **最小化到托盘**：点击窗口最小化按钮，程序会隐藏到系统托盘
- **恢复窗口**：双击系统托盘图标
- **右键菜单**：右键点击托盘图标可看到菜单：
  - 🐱 显示界面 - 恢复主窗口
  - ⏸ 暂停/播放 - 快速控制播放
  - ⏭ 下一曲 - 切换到下一首歌
  - ❌ 退出 - 完全退出程序

### 猫咪动画

界面上方有一只可爱的猫咪ASCII动画，它会根据播放状态变化：
- 🐱 **正常状态** - 猫咪在休息 (`/\_/\`)
- 🎵 **播放状态** - 猫咪在唱歌 (`> <`)
- 👂 **暂停状态** - 猫咪在认真聆听 (`o_o`)
- 💤 **空闲状态** - 猫咪会随机打瞌睡 (`-.-`)

## 🎨 主题配色

| 元素 | 颜色 | 用途 |
|------|------|------|
| 主背景 | `#FFF0F5` | 淡粉色，主窗口背景 |
| 次背景 | `#FFE4E1` | 玫瑰色，卡片背景 |
| 强调色 | `#FF69B4` | 热粉色，按钮和选中状态 |
| 深强调色 | `#FF1493` | 深粉色，悬停和激活状态 |
| 文字主色 | `#8B4513` |  saddle棕色，主要文字 |
| 文字次色 | `#CD853F` | peru色，次要文字 |

## 📁 配置文件

程序会自动生成 `player_config.json` 文件保存配置：

```json
{
  "playlists": {
    "歌单名称": {
      "folder": "/path/to/music/folder",
      "songs": [
        ["song.mp3", "/full/path/to/song.mp3"]
      ]
    }
  },
  "last_playlist": "上次播放的歌单名称"
}
```

## 📝 注意事项

- 支持的音频格式：MP3、FLAC、WAV、OGG、M4A、AAC
- APE 格式暂不支持，建议转换为 FLAC 或 MP3
- 歌单只保存文件夹路径和排序信息，不会复制音乐文件
- 如果文件夹被移动或删除，歌单会提示路径不存在
- 系统托盘功能需要 Windows 系统支持
