## ✨ 功能特性

🐱 **可爱猫咪主题**
- 粉色系配色，少女心爆棚！
- 动态猫咪ASCII动画，根据播放状态变化表情
- 圆角设计，温馨可爱

📁 **文件夹歌单**
- 一键选择文件夹，自动加载所有MP3文件
- 显示歌曲数量和列表

▶️ **播放控制**
- 播放/暂停、上一曲、下一曲
- 带有猫咪爪印风格的大按钮

📋 **播放列表**
- 右侧显示所有歌曲
- 双击即可播放

🔁 **多种播放模式**
- 列表循环 🔁
- 单曲循环 🔂
- 随机播放 🔀

🎚️ **音量控制**
- 滑动条调节音量
- 带猫咪图标的可爱设计

⏱️ **进度条**
- 实时显示播放进度
- 拖动可跳转到指定位置

🎼 **歌曲信息**
- 自动读取MP3标签（歌手、标题）

🖥️ **系统托盘**
- 最小化到系统托盘
- 双击托盘图标恢复窗口
- 右键菜单可控制播放/暂停、下一曲、退出

## 📦 安装依赖

```bash
pip install -r requirements.txt
```

依赖项：
- `customtkinter` - 现代化的Tkinter主题
- `pygame` - 音频播放
- `mutagen` - MP3元数据读取
- `Pillow` - 图像处理
- `pystray` - 系统托盘图标

## 🚀 运行程序

```bash
python music_player.py
```

## 📖 使用说明

### 基本操作

1. **添加音乐**：点击右侧的"📁 选择音乐文件夹"按钮，选择包含MP3文件的文件夹
2. **播放歌曲**：在播放列表中双击歌曲即可播放
3. **控制播放**：使用底部按钮控制：⏮ 上一曲 | ⏸/▶ 播放/暂停 | ⏭ 下一曲
4. **切换模式**：选择播放模式：列表循环 / 单曲循环 / 随机播放
5. **调节进度**：拖动进度条可跳转到指定位置
6. **调节音量**：使用音量滑块调整音量大小

### 系统托盘功能

- **最小化到托盘**：点击窗口最小化按钮，程序会隐藏到系统托盘
- **恢复窗口**：双击系统托盘图标
- **右键菜单**：右键点击托盘图标可看到菜单：
  - 🐱 显示界面 - 恢复主窗口
  - ⏸ 暂停/播放 - 快速控制播放
  - ⏭ 下一曲 - 切换到下一首歌
  - ❌ 退出 - 完全退出程序

### 猫咪动画

界面上方有一只可爱的猫咪ASCII动画，它会根据播放状态变化：
- 🐱 **正常状态** - 猫咪在休息
- 🎵 **播放状态** - 猫咪在唱歌跳舞
- 👂 **暂停状态** - 猫咪在认真聆听
- 💤 **空闲状态** - 猫咪会随机打瞌睡

## 📝 注意事项

- 仅支持MP3格式的音频文件
- 建议将MP3文件放在同一文件夹中管理
- 程序会自动读取MP3的ID3标签显示歌曲信息
- 系统托盘功能需要Windows系统支持

- Pygame - 音频播放引擎
